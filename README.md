# TechnoPHILIA'26 - Official Website

## About
TechnoPHILIA'26 is the 9th National Level Poster Presentation & Project Competition organized by IEEE Student Branch (STB60211939) and ISA at MIT Academy of Engineering, Alandi(D), Pune, Maharashtra, India.

## Event Details
- **Date**: 17th March 2026
- **Venue**: MIT Academy of Engineering, Alandi(D), Pune, Maharashtra, India
- **Organizers**: IEEE Student Branch & ISA

## Website Features
- **Responsive Design**: Works on desktop and mobile devices
- **Interactive Navigation**: Home, Key Dates, Registration, Contact Us
- **Professional Styling**: Times New Roman font with modern UI elements
- **Key Dates Table**: Attractive gradient-styled table with important dates
- **Contact Information**: Faculty and student coordinator details
- **Download Brochure**: Direct PDF download functionality
- **Registration Link**: Direct link to Google Forms registration

## Technology Stack
- **Frontend**: React.js
- **Styling**: CSS3 with gradients and animations
- **Build Tool**: Create React App

## Installation & Setup
```bash
# Clone the repository
git clone https://github.com/Aryankr0711/TechnoPHILIA_2026.git

# Navigate to project directory
cd TechnoPHILIA_2026

# Install dependencies
npm install

# Start development server
npm start

# Build for production
npm run build
```
